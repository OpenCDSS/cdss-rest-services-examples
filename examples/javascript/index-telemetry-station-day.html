JavaScript Graphing Example that Links to the State of Colorado's HydroBase Web Services

This is an example of how to use JavaScript to query the State of Colorado's Web Services. It illustrates how to retrieve and display per day water storage data. It accomplishes this by using a line graph that's easy to use, and a table that's easy to view, while taking care of behind the scenes issues such as missing data values. This example uses the Chatfield Reservoir (CHARESCO), using the STORAGE parameter in Acre-Feet (ACFT) over the last year.
